This system uses {\bfseries{compile-\/time configuration}} (preprocessor \+\#ifdefs), NOT runtime configuration. Benefits:\+

✅ {\bfseries{No runtime overhead}} -\/ Disabled code completely removed ~\newline
 ✅ {\bfseries{Smaller binary size}} -\/ Only compiled code is linked ~\newline
 ✅ {\bfseries{Zero branches}} -\/ No runtime if-\/statements for disabled features ~\newline
 ✅ {\bfseries{Simple to understand}} -\/ Clear config.\+h and platformio.\+ini ~\newline
