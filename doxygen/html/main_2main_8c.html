◆ pub_timer_cb()

static void pub_timer_cb ( void * arg)
static

Sensor data publication callback.

Called periodically to read sensor data and publish to BLE Mesh network. Payload format (18 bytes): 0-1: temp_x100 (int16 LE) 2-3: rh_x100 (uint16 LE) 4-7: raw RED (uint32 LE) 8-11: raw IR (uint32 LE) 12: heart_rate (uint8) 13: spo2 (uint8) 14-15: eco2 (uint16 LE) 16-17: tvoc (uint16 LE)
